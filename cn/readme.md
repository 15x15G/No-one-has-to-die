## https://clay.io/game/chndie

《谁都不用死》（No-one has to die）是一个关于选择的游戏。你需要选择谁生，谁死，而你的选择将攸关游戏的进行。

谁将被你拯救？谁将死去？他们的生命掌握在你的手中。
